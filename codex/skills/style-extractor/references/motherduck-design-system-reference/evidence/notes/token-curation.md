# Token Curation Notes

Strong static references do not dump every discovered value.

What to keep from MotherDuck:

- values that repeat across buttons, cards, nav, and form controls
- values that define the system's temperament: `2px` radius, strong border, warm neutral backgrounds
- examples that can be rebuilt directly

What to leave out:

- third-party integration variables that do not shape the visual language
- one-off values with no semantic reuse
- branding copy disguised as design guidance
